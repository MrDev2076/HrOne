# HrOne
Human Resource Management System for medium organisations to maintain productivity and work Synchronization
## HrOne application
**Feature -1 :** ROLE BASED ADMIN PANEL (SQL DB)
**Feature -2 :** Announcements (SQL DB)
**Feature -3 :** Meetings with participants authorisation & chatbox (sockets)⚠️
**Feature -4 :** Contact Admin (mailing service)
**Feature -5 :** Leave Request Form ⚠️
**Feature -6 :** Task Management & Workflows  (Agile - Scrum)
